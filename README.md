# NexaFlow AI – Intelligent Data Automation Platform

A premium, responsive SaaS landing page built for the **Hackathon Round 1 Challenge**. The project showcases a modern AI-powered data automation platform with a strong focus on performance, responsive design, accessibility, SEO, and interactive UI components.

# 📖 Project Overview

NexaFlow AI is a modern SaaS landing page demonstrating an enterprise-grade AI automation platform capable of intelligent data orchestration, autonomous workflows, and real-time analytics.

The project was developed following the official hackathon guidelines while emphasizing:

* Premium UI/UX
* Responsive layouts
* SEO optimization
* Accessibility
* Performance optimization
* Clean semantic HTML
* Interactive pricing engine
* Responsive Bento Grid
* Mobile Accordion transformation

## Pricing Engine

Dynamic pricing system supporting:

* Monthly Billing
* Annual Billing (20% discount)
* Multiple currencies

Supported currencies:

* 🇺🇸 USD ($)

Pricing is calculated dynamically using a configuration matrix instead of hardcoded values.
